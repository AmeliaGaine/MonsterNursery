# Monster Nursery — Book 1 Scene-Beats Index

This is the table-of-contents for the 49-episode scene-beats of *Monster Nursery*, Book 1. Each episode (`ep-01.md` … `ep-49.md`) is a per-chapter scene-beat outline carrying a header comment (micro-structure, cookie, out-type, clock, loops), an episode promise, outline tags, and numbered `## Scene` blocks closing on an `## Out`. The book runs across four arcs — Arc 1 (Ep1–12), Arc 2 (Ep13–24), Arc 3 (Ep25–37), Arc 4 (Ep38–49) — alternating two micro-structures: **K&** (Kishōtenketsu, cozy/cookie-in-the-ten) and **S&S** (Scene & Sequel, tension/cookie-on-the-turn). The tables below index each episode's title, micro-structure, scene count, cookie type, out-type, and notes (✨ SET-PIECE marquee-wonder episodes, 🍲 menu/recipe episodes, room unlocks, and the four arc climaxes).
