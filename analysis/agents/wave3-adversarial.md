# WAVE 3 — ADVERSARIAL VERIFICATION & SYNTHESIS

*The part most analysis swarms skip. Runs AFTER Waves 1–2 reports exist.*

---

## Agent 3.1 — Red-Team Skeptic

**Mandate:** A hostile genre editor whose job is to *fail* the book. Attempt a
"structural kill" against each Part-4 guardrail. For each, name the single
weakest point with a quote, then rule whether it's a **KILL** (breaks reader
trust / would tank read-through) or a **fixable nitpick**.

Guardrails to attack:
1. **Power-creep without cost** — does any gain skip its proportionate new problem?
2. **Plateau middle** — is a live clock or deepening mystery always present?
3. **Cozy-stakes mismatch** — grimdark intrusion, OR conflict so toothless it bores?
4. **Unkept promises / out-of-sequence stats** — Ledger and stat contract honored?
5. **Exposition dumps** — is System lore revealed through use, or lectured?
6. **Cliffhanger cruelty** — does each ending resolve-then-open, never betray?

**Output:** a kill-or-nitpick ruling per guardrail + the single most dangerous
finding overall. A manuscript with ANY KILL cannot be called breakout until fixed.

---

## Agent 3.2 — Evidence Adjudicator

**Mandate:** Quality-control the other agents. (Run after their reports.)
- Reject any score lacking two line-level quotes; mark it `UNSUPPORTED`.
- Surface **conflicts** (e.g., Voice loves a passage Pacing flagged as a stall)
  and force them into the open rather than averaging them away.
- Check for halo effect (a great opener inflating later scores).

**Output:** a list of upheld vs. rejected verdicts + the unresolved conflicts the
Synthesizer must rule on.

---

## Agent 3.3 — Chief Synthesizer / Showrunner

**Mandate:** The only agent that sees ALL reports. Produces the single verdict.

Apply the rubric's aggregation rule (Wave 1 gates + Wave 2 ≥5/7 quorum + no
Red-Team kill) to return one of:
- **BREAKOUT-COMMERCIAL**
- **SOLID-IN-LANE** (publishable, retention-safe, not yet breakout)
- **MISPOSITIONED** (fix lane before craft)

Then deliver, in the outline-engine's own vocabulary so fixes are actionable:
1. **Verdict + one-paragraph rationale** citing the deciding agents.
2. **The breakout assets** — the 2–3 strengths to protect at all costs (quoted).
3. **Top 3–5 highest-leverage fixes** — ranked by read-through impact, each
   concrete (a beat to add, a box to recalibrate, a sag to cut).
4. **"Good → top-tier" memo** — what specifically would move the headline scores
   from 7 to 9.
5. **Confidence + caveat** — note that this is a 2-chapter pilot; flag what only
   the full 49-episode draft can confirm (arc-climax payoff, plateau-middle, Ledger payoffs).
