# Story Grid + Foolscap Editorial Report — The Monster Nursery, Book 1

A forensic developmental edit of the Book 1 outline, run as a team of editors and story
coaches, using Shawn Coyne's **Foolscap Global Story Grid** and the **Story Grid spreadsheet**
value-shift discipline. Audits the global story, acts/sequences, beats, subplots, loops, and
cadence for anything missing or incorrect, and braids the findings into a single verdict.

## Deliverables

| File | What it is |
|---|---|
| **`index.html`** | The interactive report. Open in any browser. Tabs: Overview · Foolscap Grid · Five-Commandments map · sortable/filterable Feedback Table (29 findings) · 14-slide Highlights Slideshow (← → keys). Fully self-contained — no internet required. |
| **`SUMMARY.md`** | The editorial summary document, macro → micro, with the verdict and priority order. |
| **`FEEDBACK-TABLE.md`** | The full itemized table: severity · lens · location · finding · Story-Grid principle · concrete fix. |

## The verdict in one line

Breakout-**capable**, one reconciliation decision and four obligatory-scene fixes away from
**masterclass-locked**. The voice and the controlling idea ("care is structural") are already
top-tier; the gaps are concentrated in a few Story-Grid obligatory scenes the cozy calibration
has sanded down — all fixable without touching the cozy promise.

## The three things to decide first

1. **Lock one canon.** Two divergent outlines exist; the drafted chapters follow the older one.
2. **Resolve the planted guns.** Ch1 plants a basilisk, a griffin run, a Quiet Den and a dragon
   weathervane the current (v2) design never fires. We recommend restoring the dragon and
   grafting v2's legal/grief spine onto it.
3. **Engineer real jeopardy once.** The thesis is never tested; give the climax a real, reversible
   risk averted by care.
